Clustering: k-means, Gaussian Mixtures and DBSCAN

As with previous weeks, the working environment for this week's practicals is a set of Jupyter notebooks.

Each of these notebooks should be downloaded as a text file and run using jupyter-notebook or jupyter-lab.

The notebooks can be obtained from the following links:

  • Using a variety of clustering techniques on blob data
  • Use of GMM (soft) clustering
  • Clustering applied to the MNIST handwritten digits data

You should right-click and download each file to the same directory, and then follow the advice below.

Installation

You may need to update your python packages by installing hdbscan and python-graphviz, using the following commands:

1
2
3
conda install -c conda-forge hdbscan

conda install -c conda-forge python-graphviz

Resources

The datafile needed for the practicals can be obtained from here:

  • difficult dataset for clustering;

You should download these data files to a data folder (you might need to create this first) that is placed in the same folder as where you downloaded the notebooks.

There is also a zipped folder of python support modules. This file should be unzipped and its folder should be placed in the same folder as where you placed the notebooks.
