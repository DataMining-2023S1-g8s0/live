Pre-Start Lab

Typically the Tuesday lab sessions will based on the material covered in the preceding Friday's lecture. For the first week we will focus on installing the software.

To achieve this, the aims of this lab session are:

  • Install / Verify a Suitable Python Environment
    We will use a Python 3.8 distribution and all additional modules that will be used during this course have been tested to ensure no incompatibilities at this time.

  • Use Jupyter to write a simple script where pandas is used to read a csv file
    We start the Jupyter notebook server and write a simple python script in the browser. To stop the server, press Ctrl-C and y.

  • Review of Basic Data Exploration Skills.
    We (or more accurately you) will take a small dataset based on records of server tips, and explore it using the software tool of your choice. Later we will preform the same analysis using pandas, but for now the focus is on seeing what you can see when you look at data.
