Linear Regression Using Python

This week, we consider regression having more advanced aspects:

  • categorical predictors (that need to be transformed to numeric predictors)
  • use of regularisation when some predictors are correlated and hence too similar to each other
  • use of cross-validation for hyperparameter tuning - here, we are trying to find the best value of the regularisation parameter lambda

The first example relates to the analysis of Advertising data by media channel. We covered this example in class, so you are given guidance as to how to recreate this analysis in a notebook that you develop yourself.

The second example relates to the analysis of the Credit Balance dataset. I provide a detailed notebook with examples and discussion of how to perform linear regression with categorical-valued predictors, how to use regularisation to build models (when the model coefficients indicate that the solution is unstable) and how to use cross-validation to find the optimum value of the regularisation tuning parameter lambda.
